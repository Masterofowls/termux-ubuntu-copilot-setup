# Copilot CLI Skills & Plugins

All available plugins from both built-in marketplaces.

## Install Commands

```bash
# Single plugin
copilot plugin install <name>@awesome-copilot
copilot plugin install <name>@copilot-plugins

# Browse marketplaces
copilot plugin marketplace browse awesome-copilot
copilot plugin marketplace browse copilot-plugins

# List / update / uninstall
copilot plugin list
copilot plugin update --all
copilot plugin uninstall <name>
```
